# 项目B接线表（MVP到Pro通用）

> 先按这份接线跑通，再做结构优化。所有模块必须共地。

## 1. STM32 <-> TB6612FNG

| 功能 | STM32 | TB6612FNG | 说明 |
|---|---|---|---|
| 方向1 | PB12 | AIN1 | 电机方向控制 |
| 方向2 | PB13 | AIN2 | 电机方向控制 |
| PWM | PA8 (TIM1_CH1) | PWMA | 调速 |
| 使能 | PB14 | STBY | 必须拉高 |
| 地 | GND | GND | 共地 |

## 2. TB6612FNG <-> 电机/电源

| 功能 | TB6612FNG | 连接 |
|---|---|---|
| 电机输出 | A01/A02 | 直流电机两线 |
| 逻辑电源 | VCC | 3.3V |
| 电机电源 | VM | 5V（独立电源） |
| 地 | GND | 与STM32共地 |

## 3. 按键与限位（可选但推荐）

| 功能 | STM32 引脚 | 建议 |
|---|---|---|
| 左限位 | PA0 | 上拉输入，低电平触发 |
| 右限位 | PA1 | 上拉输入，低电平触发 |
| 开键 | PA4 | 上拉输入 |
| 停键 | PA5 | 上拉输入 |
| 关键 | PA6 | 上拉输入 |

## 4. 串口调试（CH340）

| STM32 | USB-TTL |
|---|---|
| PA9 (TX) | RX |
| PA10 (RX) | TX |
| GND | GND |

串口参数：`115200 8N1`

## 5. INMP441（Orange Pi Zero 3）

| INMP441 | Orange Pi Zero 3 |
|---|---|
| VDD | 3.3V (Pin 1/17) |
| GND | GND |
| SCK | PH6 (Pin 23) |
| WS | PH7 (Pin 19) |
| SD | PH9 (Pin 24) |
| L/R | GND |
