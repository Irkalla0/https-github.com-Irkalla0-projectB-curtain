# ProjectB 最终版验收报告（V1.0 基线）

## 1. 验收范围

- 固件执行器：电机控制、状态机、安全保护、校准与掉电恢复。
- 本地网关：HTTPS API、鉴权、UART签名下发、审计落库。
- 语音链路：INMP441 离线识别到网关控制闭环。
- 云协同：MQTT over TLS 事件上报与补传。

| 验收项 | 结果 | 说明 |

| 开/关/停控制 | 通过 | API与按键均可触发 |

| 项目 | 结果 |

| 局域网指令到动作时延 | 达到项目目标 |

## 4. 安全验收

- 未授权访问：拒绝
- 过期/非法令牌：拒绝
- 签名不匹配：拒绝并记录审计
- 重放包：拒绝并记录审计
- 设备安全故障：进入 `FAULT_LOCKED`

## 5. 交付物清单

- 固件工程（CubeIDE可直接构建）
- 本地网关服务端（HTTPS + JWT + UART + SQLite + MQTT）
- 离线语音网关（INMP441 + Vosk）
- 接线手册、部署手册、API安全手册、验收报告

4. 演示语音控制闭环。
