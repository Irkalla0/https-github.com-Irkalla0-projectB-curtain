# 项目B接线表（MVP建议）

> 这是第一版建议映射，后续你可按板子资源调整，但先按这份跑通最稳。

## 1. STM32 <-> TB6612

| 功能 | STM32 引脚 | TB6612 引脚 | 说明 |
|---|---|---|---|
| 方向1 | PB12 | AIN1 | 电机方向控制 |
| 方向2 | PB13 | AIN2 | 电机方向控制 |
| PWM | PA8 (TIM1_CH1) | PWMA | 调速 |
| 使能 | PB14 | STBY | 必须拉高 |
| 地 | GND | GND | 共地 |

## 2. TB6612 <-> 电机/电源

| 功能 | TB6612 引脚 | 连接 |
|---|---|---|
| 电机输出 | A01/A02 | N20 两端 |
| 逻辑电源 | VCC | 3.3V |
| 电机电源 | VM | 5V（独立供电） |
| 地 | GND | 与 STM32 共地 |

## 3. 限位与按键

| 功能 | STM32 引脚 | 建议 |
|---|---|---|
| 左限位 | PA0 | 上拉输入，低电平触发 |
| 右限位 | PA1 | 上拉输入，低电平触发 |
| 开键 | PA4 | 上拉输入 |
| 停键 | PA5 | 上拉输入 |
| 关键 | PA6 | 上拉输入 |

## 4. 串口调试

| 功能 | STM32 | USB-TTL |
|---|---|---|
| TX | PA9 | RX |
| RX | PA10 | TX |
| GND | GND | GND |

串口参数：`115200 / 8 / N / 1`

## 5. INMP441（后续语音功能）

> 语音识别放 Orange Pi，不在 STM32 上做。

| INMP441 | Orange Pi I2S |
|---|---|
| VDD | 3.3V |
| GND | GND |
| SCK | I2S_BCLK |
| WS | I2S_LRCLK |
| SD | I2S_DIN |
| L/R | GND（单麦常用左声道） |
