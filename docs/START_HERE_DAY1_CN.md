# 项目B从零启动（Day 1）

这份文档用于你今天直接开工，目标是把第一天做成“可验证”的结果。

## 0. 今天的目标（完成即合格）

1. 硬件接线完成（STM32 + TB6612 + N20 + 双限位 + 按键）
2. STM32 工程可下载运行（哪怕只先点灯）
3. 串口日志可输出（确认主循环在跑）
4. 电机可正反转（空载）

## 1. 必装软件（Windows）

1. `STM32CubeMX`
2. `STM32CubeIDE`
3. `ST-LINK 驱动（STSW-LINK009）`
4. `串口工具`（任一：MobaXterm / SSCOM / Termius）
5. `Python 3`

## 2. 第一版工程配置（CubeMX）

1. 时钟：默认 72MHz（先求稳）
2. GPIO：
   - 电机方向 2 路输出
   - 限位 2 路输入（上拉）
   - 按键 3 路输入（上拉）
3. TIM：
   - 1 路 PWM（电机速度）
4. UART：
   - 115200 8N1
5. SysTick：
   - 1ms

## 3. Day1验收动作

1. 开机串口打印 `boot ok`
2. 按键触发时串口打印对应事件
3. 手动拉高/拉低方向脚，电机能正反转
4. PWM 从 20%->60%->100% 有明显速度变化

## 4. 常见坑（先避开）

1. 电机供电和主控供电不分离导致复位
2. GND 没共地导致“看起来有信号但电机不动”
3. TB6612 的 `STBY` 没拉高
4. 限位输入悬空导致误触发

## 5. Day1产出物（建议保存）

1. 接线照片 2 张
2. 串口日志截图 1 张
3. 电机正反转短视频 1 段
4. 工程代码首次提交（`week1-day1: bringup done`）
