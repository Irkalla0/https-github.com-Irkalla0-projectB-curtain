# 从零开始清单（预算500内）

日期基准：2026-04（中国大陆常见电商价区间）

## 1. 必买硬件

| 类别 | 型号建议 | 数量 | 预算(元) |
|---|---|---:|---:|
| 主控 | STM32F103C8T6 最小系统板 | 1 | 12-18 |
| 下载器 | ST-Link V2（兼容版） | 1 | 12-35 |
| 电机驱动 | TB6612FNG 模块 | 1 | 6-15 |
| 执行机构 | N20 6V 减速电机 | 1 | 10-25 |
| 电机支架 | N20 支架 | 1 | 2-8 |
| 限位 | 微动开关（KW11类） | 2 | 2-8 |
| 按键 | 轻触按键 | 3 | 2-6 |
| 调试串口 | CH340 USB-TTL | 1 | 8-20 |
| 供电 | 5V2A 电源适配器（电机） | 1 | 10-30 |
| 原型搭建 | 面包板 + 杜邦线 | 1套 | 12-25 |
| 元件包 | 470uF 电容 + 常用电阻 | 1套 | 5-15 |
| 机械材料 | 尼龙线 + 定滑轮 + 扎带/胶 | 1套 | 20-50 |
| 本地网关 | Orange Pi Zero 3（1GB） | 1 | 120-220 |
| 网关存储 | TF卡 32GB/64GB（A1/A2） | 1 | 25-60 |
| 网关供电 | 5V3A Type-C 电源 | 1 | 15-35 |
| 语音输入 | INMP441 麦克风 | 1 | 8-20 |

合计建议：**289 - 590 元**  
控预算策略：选低价位时通常可控制在 **450-500 元** 内。

## 2. 预算压缩建议

1. 优先保证：主控、驱动、电机、限位、电源、网关、TF卡
2. 暂缓可选件：OLED、编码器、额外传感器
3. 滑轮选型：20-25mm U槽定滑轮（双轴承）
4. TF卡优先品牌卡，不买杂牌“超低价高速卡”

## 3. 必装软件（全部免费）

- STM32CubeIDE
- STM32CubeMX
- ST-LINK 驱动（STSW-LINK009）
- Python 3.11+
- Git
- balenaEtcher（烧录网关系统卡）
- Orange Pi 官方系统镜像（Debian/Ubuntu）
- Mosquitto（网关消息服务）
- SQLite（日志/规则存储）
- Vosk 模型（离线语音识别）

## 4. 最小上线顺序

1. 跑通 STM32 执行器（开关停 + 限位）
2. 加入位置控制和保护逻辑
3. 上线网关 API
4. 接入 INMP441 + 离线语音识别
5. 最后补安全认证与审计日志
