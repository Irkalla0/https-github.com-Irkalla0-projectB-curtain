# 现在就做（30分钟起步）

## 第1步（5分钟）：接线复核

- 电机电源是否独立供电（VM=5V）
- TB6612 的 STBY 是否拉高
- STM32/TB6612/电源是否共地

## 第2步（10分钟）：固件编译烧录

- 打开 `firmware` 工程
- 编译通过后用 ST-Link 烧录
- 串口打开 `115200 8N1`

## 第3步（10分钟）：动作验证

- 发送开/关/停命令
- 观察电机正反转与停止是否正确
- 记录一份日志

## 第4步（5分钟）：语音准备

- Orange Pi 接好 INMP441
- 先执行 `arecord -l`
- 设备出现后再跑语音网关
